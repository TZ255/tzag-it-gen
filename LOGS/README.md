This folder tracks context and changes for the Tanzania Adventures Group Itinery Generating System.

Files
- CHANGELOG.md — features, fixes, improvements (move Unreleased → Released when deploying)
- COMMENTS.md — rationale for important decisions and trade‑offs
- UPDATES.log — timestamped summaries per edit or instruction (append‑only)

Discipline
- For every code or docs change, update both AGENTS.md and LOGS/UPDATES.log.
- Use concise, English summaries. Include author or "AI" and ISO timestamp.
